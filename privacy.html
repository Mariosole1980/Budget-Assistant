Εισαγωγή

Η εφαρμογή Budget Assistant («εφαρμογή», «εμείς») δεσμεύεται στην προστασία των προσωπικών σας δεδομένων. Αυτή η Πολιτική Απορρήτου εξηγεί ποια δεδομένα συλλέγουμε, πώς τα χρησιμοποιούμε και ποια δικαιώματα έχετε.

🔒 Το Budget Assistant δεν πουλά, δεν μοιράζεται και δεν αδειοδοτεί τα προσωπικά σας δεδομένα σε τρίτους για εμπορικούς σκοπούς.

Δεδομένα που Συλλέγουμε

Η εφαρμογή αποθηκεύει μόνο τα δεδομένα που εσείς εισάγετε οικειοθελώς:

  • Οικονομικές κινήσεις: Ημερομηνία, ποσό, κατηγορία, σημείωση, τύπος (έσοδο/έξοδο)
  • Λογαριασμοί: Ονόματα λογαριασμών (π.χ. "Μετρητά", "Τράπεζα"), υπόλοιπα
  • Ρυθμίσεις: Γλώσσα, θέμα εμφάνισης, επιλογές εμφάνισης
  • Διαπιστευτήρια Cloud (προαιρετικά): Supabase URL και API Key, αποθηκευμένα μόνο τοπικά στη συσκευή σας (localStorage)

Δεν συλλέγουμε: Ονόματα χρηστών, email, αριθμούς τηλεφώνου, αριθμούς τραπεζικών λογαριασμών, κωδικούς πρόσβασης τραπεζών, ή οποιαδήποτε πληροφορία ταυτοποίησης.

Πού Αποθηκεύονται τα Δεδομένα

  • Τοπικά (offline): Τα δεδομένα αποθηκεύονται στη μνήμη του browser/συσκευής σας μέσω localStorage. Δεν αποστέλλονται πουθενά χωρίς τη ρητή σας συναίνεση.
  • Cloud (προαιρετικά): Αν επιλέξετε να χρησιμοποιήσετε το Supabase για συγχρονισμό, τα δεδομένα αποστέλλονται στον δικό σας Supabase server, τον οποίο εσείς ελέγχετε. Εμείς δεν έχουμε πρόσβαση σε αυτόν τον server.
🏠 Ο δικός σας Supabase = η δικιά σας βάση δεδομένων. Εμείς δεν φιλοξενούμε, δεν βλέπουμε και δεν έχουμε πρόσβαση στα δεδομένα σας στο cloud.

Κλείδωμα με PIN & Βιομετρικά

Αν ενεργοποιήσετε το κλείδωμα με PIN, το PIN αποθηκεύεται αποκλειστικά τοπικά στη συσκευή σας (localStorage), κρυπτογραφημένο. Δεν αποστέλλεται ποτέ σε κανέναν server.

Αν χρησιμοποιήσετε βιομετρική ταυτοποίηση (Face ID / Αποτύπωμα), η εφαρμογή χρησιμοποιεί αποκλειστικά το επίσημο Web Authentication API (WebAuthn) του browser σας. Τα βιομετρικά δεδομένα παραμένουν στο ασφαλές chip της συσκευής σας και δεν γίνονται ποτέ προσβάσιμα από εμάς.

Τρίτες Υπηρεσίες (Third-Party Services)

Η εφαρμογή χρησιμοποιεί τις παρακάτω εξωτερικές υπηρεσίες για τη λειτουργία της:

  • Google Fonts – Για τη φόρτωση γραμματοσειρών. Οι γραμματοσειρές μπορεί να αποθηκευτούν τοπικά (cache).
  • jsDelivr CDN – Για τη φόρτωση βιβλιοθηκών (Supabase.js, Chart.js, SheetJS). Δεν μεταφέρονται δεδομένα χρήστη.
  • Font Awesome CDN – Για εικονίδια (icons). Δεν μεταφέρονται δεδομένα χρήστη.
  • Supabase (προαιρετικά) – Μόνο αν εσείς επιλέξετε να χρησιμοποιήσετε cloud sync. Δείτε την Πολιτική Απορρήτου του Supabase.

Cookies & Τεχνολογίες Παρακολούθησης

Η εφαρμογή δεν χρησιμοποιεί cookies ούτε οποιαδήποτε τεχνολογία παρακολούθησης (tracking pixels, analytics, κ.λπ.). Δεν χρησιμοποιούμε Google Analytics, Facebook Pixel, ή παρόμοιες υπηρεσίες.

Δικαιώματά σας (GDPR)

Σύμφωνα με τον Γενικό Κανονισμό Προστασίας Δεδομένων (GDPR), έχετε τα εξής δικαιώματα:

  • Δικαίωμα πρόσβασης: Όλα τα δεδομένα σας είναι ορατά μέσα στην εφαρμογή.
  • Δικαίωμα διαγραφής: Μπορείτε να διαγράψετε οποιαδήποτε κίνηση ή να διαγράψετε όλα τα δεδομένα καθαρίζοντας τα δεδομένα του browser σας (Clear Site Data).
  • Δικαίωμα φορητότητας: Μπορείτε να εξάγετε όλα τα δεδομένα σας σε Excel (.xlsx) μέσω της λειτουργίας "Εξαγωγή Excel".
  • Δικαίωμα αντίρρησης: Δεν επεξεργαζόμαστε δεδομένα σας χωρίς συναίνεση.

Ασφάλεια Δεδομένων

Λαμβάνουμε εύλογα μέτρα για την προστασία των δεδομένων σας:

  • Όλα τα δεδομένα αποθηκεύονται τοπικά στη συσκευή σας.
  • Η επικοινωνία με το Supabase γίνεται μέσω HTTPS (κρυπτογραφημένη σύνδεση).
  • Τα API keys αποθηκεύονται μόνο στο localStorage της συσκευής σας.
  • Υποστηρίζεται κλείδωμα εφαρμογής με PIN 4 ψηφίων και βιομετρική ταυτοποίηση.

Ηλικιακό Όριο

Η εφαρμογή απευθύνεται σε χρήστες ηλικίας 16 ετών και άνω. Δεν συλλέγουμε εν γνώσει μας δεδομένα από παιδιά κάτω των 16 ετών.

Αλλαγές στην Πολιτική

Μπορούμε να ενημερώσουμε αυτή την Πολιτική Απορρήτου κατά καιρούς. Η ημερομηνία "Τελευταίας ενημέρωσης" στην κορυφή της σελίδας θα αντικατοπτρίζει τις αλλαγές. Συνιστούμε να επισκέπτεστε αυτή τη σελίδα περιοδικά.
